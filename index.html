⚙️ 탱커 우선순위 편집 ▶
우선순위는 기본 속성 카테고리(공격/방어/치명/...) 단위로 설정합니다.
"공격"을 1순위로 두면 공격 엠블럼과 공격보너스 엠블럼 모두 1순위 가중치로 평가되고,
둘 중 실제 Δ최종값이 큰 쪽이 자동 선택됩니다.
🔢 내 최종 스탯 & 목표값 ▶
본인 캐릭터의 최종값과 보너스%를 입력하세요. 기본 스탯은 최종 ÷ (보너스% ÷ 100)로 자동 계산됩니다.
예: 공격 17338, 공격보너스 195.7% → 기본 공격 ≈ 8859
아래 목표값은 최소 기준입니다. 미달 시 해당 스탯이 최우선으로 추천되고, 달성 후에는 설정한 우선순위 티어대로 평가됩니다. 0 또는 비워두면 목표 없음(우선순위 티어 그대로).
"🧮 목표값으로 자동 채움"은 설정된 목표값을 최종값 칸에 채워 넣습니다 (치명은 치명확률 기반 계산).

🛡️ 현재 장착 엠블럼 (선택) ▶
현재 장착 중인 엠블럼을 선택하면 추천 조합과 비교해서 어떤걸 교체해야 하는지와 스탯 변화량을 알려줍니다.
비워두면 추천 조합만 보여줍니다.

🚀 빠른 시작 (3단계)

  1. 역할 선택 — 위쪽에서 탱커 / 딜러 / 힐러 중 선택
  2. 🔢 내 최종 스탯 패널을 펼쳐서 게임 내 [속성 정보] 화면의 값을 그대로 입력
  3. 🎯 결과보기 버튼 클릭

이것만으로도 기본 추천이 나옵니다. 이후 목표값 · 우선순위 편집으로 더 세밀하게 맞추세요.

🧩 엠블럼 기본 개념

  • 그룹 3종: 🔺 기교 / ⬡ 지혜 / ◆ 활력 — 각 그룹마다 4슬롯
  • 고유 엠블럼: 1개만 장착 가능 · 비고유: 최대 2개까지 중복 가능
  • 급수 1~5: 숫자 높을수록 강력. 일부 엠블럼은 3급 또는 4급까지만 존재

⚙️ 우선순위 편집 패널

10개 스탯 카테고리(공격 / 방어 / 치명 / 명중 / 관통 / 회피 / HP / 치명저항 / 피해보너스 / 피해감소)에 순위를 배정합니다.

  • 같은 순위에 여러 스탯을 넣으면 동일 가중치로 취급
  • 스탯 드롭다운에서 "제외" 선택 시 "그 외"(최저 가중치)로 처리
  • 티어별 가중치: 1순위(100) → 2순위(85) → 3순위(70) → ... 점차 감소
  • 아래 작성자 추천 우선순위는 참고용이며 실제 계산에 영향 없음

🎯 목표값 = 최소 기준

"이 수치까지는 꼭 찍고 싶다"는 하한선입니다.

  • 미달 시 → 해당 스탯이 최우선 추천되어 기준까지 끌어올림
  • 달성 후에는 → 설정된 우선순위 티어대로 평가 (강등 아님)
  • 0 또는 빈칸 → 목표 없음 (우선순위 티어 그대로)
예시: 딜러가 "치명확률 80%는 필수, 나머지는 피해보너스 위주" 를 원할 때 → 치명확률 목표만 80으로, 나머지는 비워두면 됩니다.

🧮 목표값으로 자동 채움 버튼: 설정한 목표값을 최종 스탯 칸에 바로 채워 넣습니다 (치명은 치명확률 기반 자동 계산).

🛡️ 현재 장착 엠블럼 (선택 입력)

비워두면 추천 조합만 보여줍니다. 입력하면 교체 효과와 획득 상태가 표시됩니다:

✓ 보유이미 해당 급수 그대로 보유 중 — 추가 획득 불필요
🔧 급수 UP같은 이름 엠블럼 보유 중, 급수만 올리면 됨
➕ 추가보유 중이지만 개수 부족 (예: 2개 필요한데 1개만 보유)
🆕 신규해당 엠블럼 자체를 보유하지 않음 — 새로 파밍해야 함

🔢 공격평점 / 방어평점 공식

게임 내 [속성 정보] 맨 위의 값과 동일하게 계산됩니다.

공격평점 = (공격 + 치명 + 명중 + 관통) × (1 + 피해보너스% / 100) 방어평점 = (HP / 430 + 방어 + 회피 + 치명저항) × (1 + 피해감소% / 100)

최종 스탯 패널 상단에서 자동 계산되어 표시되며, 추천 조합을 적용하면 얼마로 변할지도 교체 시 예측 박스에서 미리 확인할 수 있습니다.

📈 획득 우선순위 — 어떤 엠블럼부터 구할까?

  • 추천 조합 중 아직 보유하지 않은 엠블럼만 노출
  • 🔥 공격평점 기여 / 🛡️ 방어평점 기여 — 두 컬럼으로 각각 내림차순 정렬
  • 기준: "이 엠블럼을 1개 더 획득하면 평점이 얼마나 오르는가"
  • 급수 제외 / 해제 시 실시간 재계산

🚫 급수 제외

지금 내가 만들 수 없는 급수를 "없는 셈" 처리하는 기능입니다.

  • 각 추천 카드의 🚫 N급 제외 버튼 클릭
  • 자동으로 한 단계 낮은 급수, 또는 아예 다른 엠블럼으로 재추천
  • 상단 제외 바에서 개별 ✕ 클릭이나 모두 해제 가능

💡 자주 묻는 질문

Q. 왜 3급 엠블럼(슬라임 / 고브 / 예티 등)은 거의 추천되지 않나요?

4급 이상 엠블럼만으로 슬롯을 채울 수 있으면 3급 천장 엠블럼은 자동으로 후순위로 밀립니다. 4급 이상 후보가 부족한 극단적 경우에만 폴백으로 추천됩니다.

Q. 설정을 바꿨는데 결과가 바로 반영되지 않아요

성능을 위해 자동 재계산은 하지 않습니다. 설정 변경 시 🎯 결과보기 버튼이 주황색 펄스로 바뀌며, 버튼을 다시 눌러야 재계산됩니다.

Q. 우선순위와 목표값 중 뭐가 우선인가요?

목표값(최소 기준) 미달 상태가 가장 강력합니다. 미달인 스탯이 무조건 최우선으로 올라갑니다. 달성 후엔 우선순위 티어 설정에 따라 평가됩니다.

Q. 현재 장착 엠블럼을 입력하지 않아도 쓸 수 있나요?

네, 비워두셔도 추천 조합은 정상 계산됩니다. 다만 ✓보유 / 🔧급수 UP 같은 획득 상태 태그와 정확한 교체 효과 수치는 나오지 않습니다.

초 평타는 1초당 1회로 가정합니다.
⚔️ 스킬 세팅 (기본/코어/전술/궁극) ▶
기본스킬은 모든 클래스 공통으로 쿨타임 1초 고정. 피해배율·피해수·지속시간만 입력합니다.
일반 스킬은 기본쿨타임·쿨감%(합연산)·피해배율·피해수·지속시간 입력. 룬 쿨감/공속은 스킬에 적용되지 않습니다.
몬스터 1마리당 한 번만 피해를 주는 스킬이면 피해수 = 1, 즉시 피해는 지속시간 = 0으로 두세요.
실효쿨타임 = 기본쿨타임 × (1 − 쿨감/100). 입력값은 자동 저장됩니다.
메인 딜 스킬 최적 조합 자동 탐색 시 이 스킬의 DPS를 최대화하는 룬 조합을 찾습니다.
기본 피해량: 캐릭터 최종 공격력 베이스. 모든 피해 계산의 기준.
모션 딜레이: 코어/전술/궁극 스킬 사용 시 그 시간만큼 다른 행동 불가 → 평타 손실. 우선순위는 궁극 → 전술 → 코어 → 기본 순으로 사용 (모든 쿨 대기 시).
🪬 룬 풀 (코어 1 / 서포트 4 선택) ▶
직접 카드를 클릭해서 코어 1개 + 서포트 4개를 선택하면 그래프가 즉시 갱신됩니다.
